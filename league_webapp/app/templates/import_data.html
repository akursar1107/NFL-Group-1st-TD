Import Historical Data

Download and cache NFL data from nflreadpy for historical analysis. This will fetch schedule, play-by-play, and roster data for the selected season.

Select Season to Import
What Gets Imported
  • Schedule Data: All games, dates, teams, and results
  • Play-by-Play Data: Every play from every game (used for first TD analysis)
  • Roster Data: Player names, positions, and team affiliations

Imported data is cached locally to speed up future requests. Re-importing will download fresh data and update the cache.
